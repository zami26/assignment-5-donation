Frequently Asked Questions

What is Document Object Model (DOM)? ⌄

The Document Object Model (DOM) is a programming interface that represents a web page as a structured tree of objects. It allows programming languages like JavaScript to access, modify, and manipulate the content, structure, and style of HTML or XML documents.

How do you select on element from the DOM? ⌄

You can select an element from the DOM using JavaScript methods like getElementById, querySelector, or getElementsByClassName. For example, document.getElementById("title") selects the element with ID title. These methods let you access and manipulate HTML elements dynamically.

What is event delegation in the context of DOM, an why is it useful? ⌄

Event delegation is a technique where a single event listener is added to a parent element to handle events on its child elements using event bubbling. It’s useful because it improves performance and simplifies code by avoiding multiple listeners, especially for dynamic or large lists of elements.

How do you manipulate an element's attributes and style using the DOM? ⌄

You can manipulate an element's attributes using methods like setAttribute, getAttribute, and removeAttribute. To change styles, use the .style property—for example, element.style.color = "red".
